# template_clean_code

### Autor
Guilherme Eduardo Walter

### Descrição
Projeto template para estudo de clean code criado na capacitação técnica do Senac

### Tecnologias usadas
- Node

### Estrutura de pastas
template_clean_code/
├── database
├── docs/
├── src/
│   ├── models/
├── public/
│   ├── assets/
├── tests/
└── README.md

- database: contém as configurações do banco de dados
- docs: arquivos de documentação
- src: arquivos de código da aplicação
- public: arquivos públicos acessíveis pelo site da aplicação
- tests: arquivos de testes da aplicação

### Final
Este é um TPMV no VSCode. É simples, mas é a fundação para a qualidade.
